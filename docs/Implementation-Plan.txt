# 🧱 Phase 0 — Project Setup (Day 1)

## Backend

* Initialize Node.js project
* Install dependencies:

  * express, mongoose, dotenv
  * jsonwebtoken, bcrypt
  * socket.io, cors
* Create folder structure (as per architecture)

## Frontend

* Create React app (Vite recommended)
* Install:

  * react-router-dom
  * axios
  * tailwindcss
  * socket.io-client

## Database

* Setup MongoDB (local or Atlas)
* Connect using mongoose

---

# 🔐 Phase 1 — Authentication System (Day 2–3)

## Backend

* Implement User model
* Create:

  * register API
  * login API
* Add JWT auth
* Add password hashing (bcrypt)

## Frontend

* Build Login page
* Store JWT + role
* Implement redirect based on role

---

# 🧩 Phase 2 — Role-Based Routing & Layout (Day 3–4)

## Frontend

* Create layouts:

  * Student
  * Admin
  * Patrol
  * Super Admin

* Implement ProtectedRoute

* Setup routes:

  * /student
  * /admin
  * /patrol
  * /super-admin
